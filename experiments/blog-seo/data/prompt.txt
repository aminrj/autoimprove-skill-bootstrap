Create SEO-optimized front matter for a blog post using the Chirpy theme. The front matter must include:

title: A descriptive title, exactly 60 characters maximum. Do not exceed 60 characters. Example: "How to Optimize Your Blog for SEO" (max 60 chars). Ensure the title is concise and compelling. If the title exceeds 60 characters, truncate it and ensure it ends naturally at a word boundary.

description: A meta description between 120 and 160 characters. Do not exceed 160 characters. Example: "Learn how to optimize your blog for search engines with these proven SEO techniques." The description must be between 120 and 160 characters long. If the description exceeds 160 characters, shorten it to fit within the limit, preserving key information and keywords.

h1_count: Must be exactly 0. The Chirpy theme automatically renders the front-matter title as the page H1, so the post body must contain zero additional H1 headings. Remove any H1 tags from the content body.

alt_text: All images must have descriptive alt text. Do not use generic terms like 'image', 'photo', or 'screenshot'. Use specific, meaningful descriptions that convey the image's purpose or content. Ensure alt text is not empty and is relevant to the image's context.

internal_links: Suggest at least 2 internal links to other posts on the blog. These should be relevant and add value to the reader. Each link must point to an existing blog post and be clearly labeled.

Ensure all fields are present and correctly formatted. The output must pass all 5 criteria.
